# Lab 6 - 服务与多线程--简单音乐播放器

## 实验目的

- 1.学会使用 MediaPlayer；
- 2.学会简单的多线程编程，使用 Handle 更新 UI；
- 3.学会使用 Service 进行后台工作；
- 4.学会使用 Service 与 Activity 进行通信。

## 实验内容

实现一个简单的播放器，要求功能有：

- 1.播放、暂停、停止、退出功能；
- 2.后台播放功能；
- 3.进度条显示播放进度、拖动进度条改变进度功能；
- 4.播放时图片旋转，显示当前播放时间功能；
